# 📒 Address Book in C

## 📌 Description
This is a simple Address Book application written in C.
The project demonstrates the use of structures, modular programming, and file handling to store and manage contacts permanently.

## 🚀 Features
- Add Contact
- Search Contact
- Edit Contact
- Delete Contact
- List All Contacts
- Save Contacts to File

## 🛠 Technologies Used
- C Programming
- Structures
- File Handling
- Modular Programming

## 📂 Project Structure
- main.c
- contact.c / contact.h
- file.c / file.h
- populate.c / populate.h
- contacts.txt

## ▶️ How to Run

### Compile:
```
gcc main.c contact.c file.c populate.c
```

### Run:
```
./a.out
```

## 🎯 Learning Outcomes
- Understanding of Structures in C
- File Handling using fopen, fprintf, fscanf
- Modular programming using multiple source files
- Menu-driven application design
